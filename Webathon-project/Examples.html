LANGUAGE EXAMPLES

Python

1.This is just a basic Syntax for Printing hello world:

print("Hello World")

2.This is just a basic Syntax for using for loop

for i in range(startvalue,end value): #where i is a iterator variable

3.This is a basic syntax for using while loop:

While(condition):#it will continue to iterate untill the satisfying condition is executed

Language Description:

It is a dynamic programming language which does not require any specific header files for code execution and python by default takes all inputs in form of string.

Python

1.This is just a basic Syntax for Printing hello world:

printf("Hello World");

2.This is just a basic Syntax for using for loop

for(i=startvalue;conditionincrement/decrement){} //where i is a iterator variable

3.This is a basic syntax for using while loop:

While(condition){}//it will continue to iterate untill the satisfying condition is executed

Language Description:

It is the most powerful programming language which requires a header file #include stdio.h within <> with specified brackets and the data type of the variable must be specified during declaration.
